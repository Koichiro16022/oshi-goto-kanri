# 推し事管理アプリ 開発メモ

## アプリ概要
推し活をしている人向けのPC用タスク管理HTMLアプリ。
ブラウザで動作する単一HTMLファイル。PC用Chrome/Edge/Firefox推奨。

https://booth.pm （ショップ登録・商品出品済み）

## 同梱ファイル構成（販売ZIPに含めるもの）
- `推し事管理.html` … メインアプリ本体
- `icon.png` … アプリアイコン（192×192px）。base64埋め込みは不可。別ファイルとして同梱する
- `推し事管理_使い方ガイド.pdf` … 使い方ガイド。スクリーンショット挿入後に最終確認予定

## 設計方針
- タスクは全推し共通で一括管理
- 推しごとにイメージカラーを設定しタスクの左ライン（border-left）のみに反映
- 背景色は推しカラーとは独立したカスタマイズ設定（推しカラーは背景色に使わない）
- 画像は全推し共通・シャッフルキュー方式で表示（全画像を1回ずつ表示してループ、最大10枚）
- シンプルさを優先。必要性を感じた機能だけ追加する
- スマホ対応は要望があれば検討

## 実装済み機能
- タスクの追加・修正・削除・完了アーカイブ
- 推し管理（最大5つ）：名前・イメージカラー・プリセット9色スウォッチ、その場で編集可能
- 推し管理の表示順を▲▼ボタンで入れ替え可能（先頭・末尾は薄表示で操作不可）
- 並び替え時にタスク・アーカイブの推しカラー参照（oshiIdx）も自動更新
- 推しカラーのリアルタイム反映（カウントダウン・アクティブタスク両方）
- メインイベントのカウントダウン表示
- リマインダー通知（再通知間隔・上限回数設定）／おやすみモード
- システム通知対応（Notifications API）：バックグラウンド時はシステム通知、フォアグラウンド時はシャワーエフェクト＋音も併用
- 推し画像コレクション（最大10枚、シャッフルキュー方式）
- 非画像ファイル登録時のエラーアラート
- タスク完了時の褒め言葉エフェクト（花火・シャワーワード・推し画像）
- 褒め言葉・リマインダーセリフの編集（初期値20種類登録済み）
- データのエクスポート（設定を保存する）／インポート（設定を読み込む）
- 画像リセット／タスクリセット（独立した初期化ボタン）
- ヘルプパネル（ヘッダーの「？ ヘルプ」ボタンから7項目表示）
- アプリアイコン（icon.png をファビコン・Apple Touch Icon として設定）
- index.html を追加（GitHub Pages での URL 直アクセス対応）

## 作業ログ

### 2026-05-08
- **システム通知対応（Notifications API）**
  - 起動時に `Notification.requestPermission()` で通知許可をリクエスト
  - リマインダー発火時に `new Notification(タスク名, { body: セリフ, icon: 'icon.png' })` を送信
  - `document.hidden`（Page Visibility API）でフォアグラウンド判定
    - バックグラウンド時：システム通知のみ
    - フォアグラウンド時：システム通知 ＋ シャワーエフェクト ＋ 音
  - おやすみモードは引き続き両方をスキップ
  - HTTPS / localhost 環境でのみ動作（コメントに明記）
- **再通知間隔に「1分ごと」を追加**
  - セレクトボックスに `<option value="1">1分ごと</option>` を追加（なし・5分ごとの間）
